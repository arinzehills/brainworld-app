# Brain World Mobile APP

This is a brainworld mobile written in flutter. it's is mobile e-learning app.

##App functionalities##

- **Purchase Courses**
- **Purchase Books**
- **Save personal Books**
- **Chat system**
- **Video conferencing/Virtual classrooms**
- **Video call**
- **Voice internet call**
